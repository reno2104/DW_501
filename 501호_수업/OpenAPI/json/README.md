### JSON(JavaScript Object Notation)

    데이터를 저장하거나 전송할 때 많이 사용되는 경량의 DATA 형식.

### 특징

1. 키와 값으로 구성.
2. 특정 언어에 종속되지 않으며, 대부분의 프로그래밍 언어에서 JSON 지원.

### 문법

```json
// JSON도 데이터 타입이 존재.
// null, number, string, array, object, boolean을 사용할 수 있음.
{
  "empno": 7369,
  "ename": "SMITH",
  "job": "CLERK",
  "mgr": [7902, 7369],
  "sal": 1200,
  "dname": "SALES"
}
```
